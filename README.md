# Power-Bi-Project
Project name :- Quality Audit Performance Dashboard 
To analyze employee performance, defects %, errors %, and overall quality score using interactive dashboards.
Analyze overall quality performance
Track Defects % and Errors % by month
Compare performance across:
Departments
Managers
Employees
Locations
Identify low-performing areas
Support data-driven decision making.

# 📊 Quality Assurance Dashboard | Power BI

An interactive **Quality Assurance Dashboard** built using **Microsoft Power BI** to monitor quality performance, defect rates, error rates, auditor performance, departmental quality, and employee-level quality metrics.

---

## 📌 Project Overview

This dashboard helps organizations track quality KPIs, identify performance gaps, monitor defects, and evaluate employees, auditors, managers, and departments through interactive visualizations.

The report enables management to make data-driven decisions to improve operational quality and reduce defects.

### 📈 Executive KPI Overview

- Sample Total

- Defect Percentage
- Error Percentage

### 📊 Monthly Performance Analysis

- Monthly Defect %
- Monthly Error %
- Defect vs Error Comparison
- Performance Trend Analysis

---

### 👥 Employee Performance

- Employee Quality Score
- Sample Count
- Defect Rate
- Error Rate
- Employee-wise Quality Analysis

---

### 🏢 Department Analysis

Analyze quality performance across departments such as:

- Back Office
- Finance
- Sales

---

### 👨‍💼 Manager Performance

Evaluate quality metrics by manager including:

- Quality Score
- Sample %
- Defect %
- Error %

### 🧑‍💼 Auditor Performance

Compare auditor performance using interactive filters.

Supported auditors include:

- Auditor A
- Auditor X

- Auditor Z

Track quality metrics across office locations to identify regional performance differences.
